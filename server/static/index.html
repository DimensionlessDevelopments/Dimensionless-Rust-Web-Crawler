About Rust Link Crawler

Rust Link Crawler is a high-performance web crawler built with Rust, Axum, and reqwest. We combine the speed and safety of Rust with modern web frameworks to deliver seamless link checking and validation.

Powered by reqwest and scraper, our crawler can traverse websites, discover links, and check their status codes. Whether you need to validate internal links, find broken pages, or map site structure, Rust Link Crawler is here to help.
